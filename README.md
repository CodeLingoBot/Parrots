 Parrots
========

 Introduction
--------------
Simple web service that outputs what it has been given as input.

 Installation
--------------

Clone this repository and run `composer install`.

 Usage
----------------

Visit the page in a browser, add something after the main URL, be amazed by the
result.

----------------
